How to Send a shipment


    Delivery Method

    Add a delivery method to the orde.

    Put in Pack

    When the Sales order items are "Done", you need to add them to a "pack".

    If you want several "packs" (boxes etc), then add all items to be placed in the first box (by selecting "done"), then press "Put in pack". Then add all items for the second "pack" and press "put in pack" for the second "pack" etc.

    Select type of Package

    Select the type of package to be used.

    Validate to create a Packing label

    When you click Validate, the app will send a request to Bring and return a Packing Label.

    Shipping label

    Print the shippinglabel from the pdf in the "chatter"-sectio.

    How to Return Goods


    Retrun Items

    Go to the Sales-Order where the goods were sold. Click on "Delivery"

    Click on "Return" to initiate a Return process.

    Click on the "Burger" to update the "done"-quantity. Then Click confirm.

    Add the return "Carrier", in this case what "Shipping method" should be used.
    Finally, Click "Validate" -> A PDF-return-label should appear in the right window.
